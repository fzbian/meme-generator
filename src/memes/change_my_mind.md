# Change My Mind Meme

## API Reference

```http
  GET https://grasapi.fzbian.me/api/change_my_mind/
```

## Params

|Parameter|Type|Description|
|---|---|---|
|`text`|`string`|the text that will go on the banner|

## Example
```http
  GET https://grasapi.fzbian.me/api/change_my_mind/?text=pineapple%20pizza%20sucks
```
